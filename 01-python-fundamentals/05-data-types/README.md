# 🧩 Data Types in Python

## Overview

Data types tell Python what kind of value a variable stores.

Python provides several built-in data types. The most commonly used are:

- Numbers
- Strings
- Booleans

Understanding these data types is essential because every variable in Python stores one of these values.
